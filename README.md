# Music-Server
compose stack for the container services which manage my music collection, as well as act as a backend for my further music based projects.
only the .env with your slskd login information is needed to gain file network access, other than that it should just run. 

## Access Points
Navidrome → http://localhost:4533

slskd → http://localhost:5030

## .env structure
SLSKD_USERNAME=xyz
SLSKD_PASSWORD=xyz
